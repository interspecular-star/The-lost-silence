# Руководство автора The Lost Silence

Для владельца: как собирать игру самостоятельно, без программиста.
Все механики ниже уже работают — документ говорит, **куда нажать** и по каким
рецептам собираются типовые вещи. В конце — честный список «сырых мест»:
что пока нельзя настроить из редактора и как это обойти.

Проверяй себя двумя кнопками: **«✓ Проверка»** (ловит битые ссылки и опечатки,
клик по строке — переход к месту) и **«▶ Играть»** (F5; там же выбор стартовой
сцены и чекпоинты «⚑» — можно начинать плейтест с любого места).

---

## 1. Сцены, папки, слои

- Сцена = фон + элементы (текст / панель / изображение / кнопка / зона клика).
  Холст 1920×1080; в игре всё масштабируется само, и на телефоне тоже.
- **Папки** (сайдбар слева): строка «📁 Новая папка (глава)» создаёт папку;
  у сцены меню ⋯ → «📁 В папку…». Меню папки: переименовать, **дублировать со
  сценами** (переходы между сценами внутри папки автоматически ведут на копии,
  имена получают «(копия)»), расформировать (сцены наружу), удалить со сценами.
  «+» в папке создаёт сцену типа «локация» — тип меняется в инспекторе.
  Папки — только для порядка в редакторе, игра их не видит.
- Порядок сцен — перетаскиванием за ⠿ (внутри одной группы/папки).

### Анатомия локации (наш стандарт, смотри «Двор лагеря» как образец)
1. **Кикер** — текст капсом с разрядкой сверху слева: `АВАНПОСТ · ДВОР`.
2. **Нарратив** — 2–4 абзаца (пустая строка = новый абзац).
3. **Точки** — кнопки-строки: `◊` люди/важное (часто бирюзовым #4fd1c5),
   `·` объекты/осмотр, `▲` риск, `[Сила]`/`[Ловкость]` — проверки.
4. **Выход** — «‹ К СХЕМЕ ЛАГЕРЯ» внизу слева, всегда в одном месте.

## 2. Карта лагеря (план с ромбами)

Панель «Мир» при выбранной сцене → секция **«Карта лагеря»** → галка
«эта сцена — карта». Дальше:
- **Узел** = ромб: название, «Ведёт в сцену» (кнопка ВОЙТИ; повторный клик по
  ромбу — тоже вход), X/Y в % (центр ромба), Размер (ширина в %, обычные 12–16,
  дальние 8), «Даль» (приглушение 0–100 для заднего плана).
- **Примета** — строка в сайдбаре узла («сварка и лебёдки»).
- **Живые пометки** — строки-компасы на самом ромбе: показывается **первая,
  чьи условия верны**. Порядок важен: условные — выше, безусловную-фон — последней.
  `◊` в начале текста красит пометку в акцент, прочее — приглушённо.
- **Кто здесь** — список NPC для сайдбара (портрет + роль из карточки).
- **Заперто** — условия + текст отказа (ворота: `gates_open = нет` →
  «Караульный качает головой…»). Запертый узел без сцены — нормально.
- **Положение ГГ** — какой узел пульсирует и пишется в «ТЕКУЩЕЕ ПОЛОЖЕНИЕ»
  до первого входа куда-либо; дальше движок помнит сам (и в сейве).
- **Дорожки** — пары узлов, пунктирные линии.

Обычные элементы сцены (кикер-заголовок) рисуются ПОД картой — заголовок карты
делай текстовым элементом. Холст показывает план статично, поведение — в F5.

## 3. HUD и Осколок

У сцены поле **HUD**: Авто (скрыт на страницах) / Показывать / Скрывать /
**«С Осколка (ур. 1+)»** — режим лагеря и вылазок: HUD появляется в ту же
секунду, когда эффект ставит `oskolok = 1`.

Лестница Осколка (переменная `oskolok`, 0–10, канон в `oskolok-mesh.md`):
уже работают **1** (шкала отношения в диалоге + тумблер MESH в HUD),
**2** (панель фракций ◈), **3** (подсказки ▲▼ на вариантах), **4** (боевой скан),
**6** (страхи/желания в профилях персонажей), **8** (расшифровка OldNet ×2 быстрее).
Уровни 5 / 7 / 9 / 10 зарезервированы — пока ничего не включают.
Апгрейд = просто эффект «oskolok = N» в нужном месте сюжета.

**Тумблер MESH** появляется в HUD сам, если есть переменная `mesh_on` и
oskolok ≥ 1. Выключенный Mesh глушит канал шёпотов целиком.

## 4. Шёпоты Архона (полоса над HUD)

Режим «Журнал» → секция «◈ Шёпот Архона». У шёпота: текст (панель стилей
работает, прелесть даёт `[type]…[/]` — печать), **триггер**:
- «вход в сцену» (+какая), «конец диалога» (+какой), «время тишины» (idle,
  повторяемый + кулдаун в минутах), «вручную» (нода «Действие» → «◈ Прошептать»
  или ответ на чип).
- условия, задержка, сколько висит, «важный» (вперёд очереди).
- **до 3 чипов-ответов**: текст + эффекты + **ответный шёпот** — так собираются
  целые разговоры с Архоном. Молчание игрока — тоже ответ (движок сам считает
  `mesh_ignored`; выборы — `mesh_answered`).

**Образец — «Первый контакт»** (4 шёпота цепочкой): диалог сборки кончается
эффектом `oskolok = 1` → HUD проявляется → шёпот с триггером «конец диалога»
печатает «Здравствуйте.» → чип «Кто ты?» ведёт в ответный шёпот, и так далее.
Такой же схемой делается любой разговор с Архоном без диалогового окна.

Правила вежливости встроены: в бою канал ждёт, поверх диалога полоса
приглушается, `mesh_on = нет` глушит всё. Глиф ◈ справа — перечитать
последний шёпот; журнал голоса — вкладка «◈ ГОЛОС» в 📋.

## 5. Зоны аномалий (таймер экспозиции)

Панель «Мир» → секция **«Зона аномалии (таймер)»**:
- Экспозиция (сек) — безопасное время; на экране идёт «счёт про себя»
  прописью (диегетика: интерфейса у ГГ нет).
- Урон HP/с после истечения (счёт сменяется красным «сбился со счёта…»),
  HP не падает ниже 1; реген HP в зоне выключен.
- Откат (сек): вышел — идёт откат; вернулся раньше — бюджет продолжается
  с остатка; после отката — полный.
- «Выброс при HP ≤ %» + **аварийные выходы**: список «условия → сцена»,
  срабатывает первый истинный. Рецепт 1.4: выход «спас с верёвкой» по флагу
  `ch1_temur_helped`, последним — выход по умолчанию (без условий).

Атмосфера зоны — «Эффекты фона» с условиями (глитч/виньетка по флагу пути).

## 6. Точки обыска и «Осмотреться» (рецепты)

**Точка со случайным лутом** (мини-диалог, образец «Куча корпусов»):
1. Нода «Действие»: флаг одноразовости `= да` + эффект **«🎲 1..N»**
   в переменную `ch1_rnd` (N = число исходов).
2. Нода «Условие»: `ch1_rnd ≤ 2` → ветка «истина» (лут), «ложь» (пусто).
   Для трёх исходов — второе условие в ветке «ложь».
3. В ветках: реплика-курсив (`[i]…[/]`, без спикера) + «Действие» с
   «Выдать предметы». Кнопка на сцене: «Показывать при условии» флаг `= нет` —
   точка исчезает после обыска. Шансы «30/70» — тот же 🎲 1..10 и порог ≤ 3.

**«Осмотреться» со счётчиком** (образец — кладбище): «Действие» `+1` к
счётчику → лесенка «Условий» (`≤1` → строка 1; `≤2` → строка 2; …) →
на нужном шаге выдай предмет/открой скрытую точку (её `visibleIf: счётчик ≥ 3`).

**Проверки характеристик**: кнопка «[Сила] …» с условием видимости
`atk ≥ порог` (базовые значения героя — режим «Предметы»: сила 10, ловкость 5).

## 7. Квесты-цепочки и компасы

Режим «Журнал»: у задания — **этапы** (текст + условия; проходятся по порядку,
навсегда). Условия этапов вешай на флаги (`ch1_kai_list = да`), флаги ставь
эффектами в диалогах. Награда забирается в журнале 📋 — помни: до Осколка
журнала нет, игрока ведут **пометки на карте** («◊ к Каю, за списком») и
напоминалки — делай их всегда.

**Напоминалки**: несколько кнопок на одном месте сцены с разными `visibleIf`
по стадиям (знакомство → сюжетная точка → напоминание) — следи, чтобы условия
не пересекались, иначе кнопки лягут друг на друга.

## 8. Предметы: выдать, забрать, продать

Нода «Действие» (и действия кнопок) умеют «Выдать предметы».
Только нода «Действие» дополнительно умеет:
- **«Забрать предметы»** — кол-во, `0 = все, сколько есть` (сборка Осколка
  забирает катушку/шину/ящик);
- галка **«начислить цену забранного в валюту»** — продажа по цене предмета
  (сдача хлама Каю). Цены — в карточках предметов, валюта — `credits`.
Квестовые предметы (галка в карточке) нельзя продать/выбросить из инвентаря.

## 9. Бои и «стены»

Режим «Мобы»: HP/атака/защита/замах/криты/атаки с весами/награды/дроп.
**Мрека-стена** (падальщик): HP 9999, урон 999, замах 150 мс — окно уклона
ограничено 70% замаха, короткие замахи честно неуворачиваемы («✓ Проверка»
предупредит о замахе < 400 мс — для стены это норма, для обычного моба — нет).
Запуск боя — действие кнопки «Начать бой» (+ диалоги победы/поражения;
поражение ставит HP = 1). Приём «ретроспектива»: бой мгновенный, а ЧТО
произошло — рассказывает диалог поражения, в конце нода «Переход» уводит
на сцену «очнулся». Скан в бою — с Осколка ур. 4.

## 10. Материалы и голос Mesh в диалогах

Библиотека материалов — панель «Мир». Реплика получает материал в ноде
(селект «Материал»), приоритет: реплика > правила диалога > диалог > NPC >
сцена > фракция > тема. Для Архона в диалоге: спикер-текст `Mesh` (обычное
поле «Спикер», это не NPC) + материал «Архон / Mesh» + текст в `[type.once]…[/]`.
Но канон теперь: **Архон говорит шёпотами**, диалоговое окно — только людям.

## 11. Титул главы (рецепт)

Страница с тёмным фоном → текст «ГЛАВА N» (появление «Проявление», задержка
0.5) → текст названия (появление «Из размытия», задержка ~1.1) → в панели
«Мир»: «Автопереход в …» через ~5 с, «Фейд ухода» 1.5–2. Вход в титул —
тихая строка на сцене с условиями (как «· Дни идут своим чередом»).

## 12. Справочник переменных главы 1

`oskolok` 0–10 · `mesh_on` тумблер · `mesh_answered/mesh_ignored` счётчики голоса ·
`gates_open` ворота · `credits` валюта.
Флаги сюжета: `ch1_started` (титул пройден) · `ch1_liya_brief` (1.1) ·
`ch1_kai_list` (1.2, открывает ворота) · `ch1_coil_found`/`ch1_bus_found` (детали) ·
`ch1_servo_found`/`ch1_servo_returned` (сервопривод) · `ch1_temur_talk`/`
ch1_temur_helped` (развилка 1.4) · `ch1_went_yellow`/`ch1_temur_saved`/`
ch1_yellow_rescued` (последствия жёлтой) · `ch1_temur_after`/`ch1_temur_keys`
(гнездо + ключи) · `ch1_nest_bait`/`ch1_nest_tried`/`ch1_scavenger_beaten_by`
(1.5) · `ch1_temur_gate` (вставка у ворот) · `ch1_kai_order` (1.6) ·
`ch1_act1_done` (финал). Счётчики осмотров: `ch1_scrap_look`/`ch1_ruins_look`/`
ch1_yellow_look`/`ch1_nest_look`; одноразовые точки: `ch1_p_*`; бросок: `ch1_rnd`.

## 13. Сохранения и экспорт

- Автосейв редактора: браузер + файл `local-save/project.json` (+20 копий в
  `local-save/history/`). Кнопка «Сохранить» — файл `.tls.json` на диск.
- Предпросмотр всегда с чистого листа (прогресс игрока не пишется);
  оффлайн-idle и сейвы игрока живут только в экспортированной игре.
- Экспорт: выбор старта (стартовая/сцена/чекпоинт); новая сборка со штампом
  не подхватывает сейвы старых сборок — игроки начнут заново.

## 14. Сырые места: чего пока НЕТ в редакторе (и обходы)

Честный список. Если что-то из этого понадобится — это заявка на код, зови.

**Карта лагеря**
- Ромбы не перетаскиваются мышью на холсте — только числа X/Y в инспекторе
  (холст сразу показывает результат, двигать удобно шагами по 2–3%).
- Пометки узла нельзя менять местами — порядок = порядок создания
  (удали и пересоздай в нужном порядке).
- Дорожки — только прямые пунктиры, без изгибов и условий.

**Зоны**
- Одна зона на сцену; урон — только по HP и только постоянный (нет нарастания).
- Счёт «про себя»: позиция/стиль/тексты («сбился со счёта…») зашиты.

**Условия и предметы**
- Условия — только «И». «ИЛИ» собирается двумя элементами/ветками с разными
  условиями или общим флагом, который ставят обе ветки.
- Условия «есть предмет N» нет — наличие отслеживай флагом, который ставится
  при выдаче (так сделан весь акт 1).
- Продажа — «всё разом по цене предмета»; частичная продажа, торг, магазин
  с витриной — это будущий блок C3.

**Шёпоты**
- Полоса не настраивается (позиция top 8.5%, цвет, шрифт); чипов максимум 3.
- На молчание нельзя повесить ответный шёпот — только счётчик `mesh_ignored`
  для условий будущих шёпотов.

**Диалоги**
- Спикер «Mesh» — просто текст: движок не знает такого персонажа (нет
  отношения/знакомства). Это осознанно: Архон — не NPC.
- Дублирование папки не дублирует диалоги: скопированные сцены запускают
  ОБЩИЕ диалоги, а «Переходы» внутри этих диалогов ведут в сцены-оригиналы.
  Для «шаблона главы» дублируй диалог отдельно (меню диалога → ⧉) и перевесь.
- Режим «Черновик» не видит ветвлений/эффектов — сложные диалоги правь в графе.

**HUD и прочее**
- Состав и позиции HUD фиксированы; валюта одна (`credits`).
- Титул главы — рецепт руками (п. 11), кнопки-шаблона нет.
- Чекпоинты плейтеста не запоминают остаток зонного таймера и «текущее
  положение» на карте — после загрузки чекпоинта они сбросятся.
- Уровни Осколка 5/7/9/10 пока пустые (разметка аномалий, следы OldNet,
  тихий канал, финал — будущие фазы).
- Ачивки/задания проверяются раз в секунду — эффект «мигом» может догнать
  уведомлением на полсекунды позже, это нормально.

**Стиль (напоминание себе)**
- Тихие строки-действия, hairline-рамки, капс с разрядкой; маркеры: ◊ люди
  и важное, · объекты, ▲ риск, ··· заперто, ◈ системное. Меньше, но точнее.
