## Сцена 1.3 — «Кладбище техники» (первая вылазка; новые механики)

Механики сцены (решения владельца, 2026-07-08): точки обыска со случайным лутом;
проверки характеристик с малыми порогами (100% успех — игрок «щупает» механику ещё
до Осколка и HUD); **«Осмотреться» с внутренним голосом ГГ** — счётчик нажатий, текст
меняется, после 3-го нажатия предмет + открывается скрытая точка. «Шум» — только
зачатки в тексте, без механики.

*Локация: кладбище техники за северными воротами. Ряды мёртвых машин, песок, ветер.
Настроение: тихо, просторно, никто не торопит.*

*Нарратив сцены:*

Кай не соврал: катушки здесь были. Лет двести назад. С тех пор кладбище пережило
три поколения сборщиков, и всё, что лежало сверху, давно унесли.

Значит, смотреть надо там, куда лень было лезть.

### Точки на сцене (элементы-действия)

1. **«Генераторный блок»** (гарантированная, для квеста): полуразобранный корпус
   в дальнем ряду. Осмотр → короткий нарратив («кто-то уже снял кожух, но до обмотки
   не добрался — руки были толще твоих») → **Резонансная катушка** получена.
2. **«Куча корпусов»** (случайная): хлам на продажу / пусто.
3. **«Опрокинутый контейнер»** (случайная): расходник / хлам / пусто.
4. **«Кабина крана»** (случайная + проверка): дверь заклинило.
   Действие «[Сила] Отжать дверь» → внутри расходник + хлам.
5. **«Штабель плит»** (проверка): «[Ловкость] Забраться наверх» → сверху видно всё
   кладбище (нарратив-обзор: где-то за рядами блеснуло и погасло — зачаток «шума»,
   намёк на падальщика) + хлам в ласточкином гнезде из проводов.
6. **«Занесённый песком люк»** (скрытая, `visibleIf: ch1_scrap_look >= 3`):
   ремонтный кэш под землёй — цел, потому что не блестел. Внутри: хлам подороже
   + расходник. Закрывает вылазку приятной жирной точкой.

### «Осмотреться» (кнопка на сцене, многоразовая)

Каждое нажатие: `ch1_scrap_look +1`, показывается строка внутреннего голоса ГГ
(реализация — мини-диалог: branch по счётчику → line без спикера, курсивом):

1. *«Под завалами — хлам. Хлам, хлам, ещё раз хлам. Двести лет как разграблено.»*
2. *«Дрон. Лежит смирно… надеюсь, не оживёт. Ладно. Не буду проверять.»*
3. *«Хм. Вон та плита просела, а песок вокруг неё — нет. Под ней что-то есть.»*
   → **предмет «Рабочий сервопривод»** (блестящая исправная деталь — пригодится
   для обмена с падальщиком в сцене 1.5!) + открывается точка 6 «Люк».
4. (и далее) *«Всё. Здесь я видел каждый болт. Пора к руинам.»*

### Технические примечания

⟦Случайный лут: нужен новый эффект движка «случайное число в переменную»
(op: 'random', 1..N) — добавить в код-пакет C7 при внесении главы. Схема точки:
осмотр → set random → branch по значению → giveItems/нарратив. Точки одноразовые —
флаг на каждой (visibleIf или условие на действии).⟧
⟦Проверки статов: условия на действиях по atk («Сила») и agi («Ловкость»), порог =
стартовое значение героя → всегда проходят. Подпись на кнопке: «[Сила] Отжать дверь».
Игрок замечает, что у него ЕСТЬ характеристики, — до всякого HUD.⟧
⟦Внутренний голос: line-ноды без speakerNpcId, курсив; счётчик ch1_scrap_look.
Паттерн переиспользуемый — записан как стандарт для будущих сцен-вылазок.⟧
⟦Шум: механики НЕТ (решение владельца — без Осколка/HUD слепо). Зачатки в тексте:
блеск за рядами (т.5), «надеюсь, не оживёт» (голос №2). Падальщик в 1.5 может
«узнать» ГГ, если тот лазил по штабелю — задел, решим при 1.5.⟧
⟦Лут-номенклатура (хлам — resource, продаётся Каю): «Обгоревшая обмотка»,
«Гнутый крепёж», «Пласт изоляции»; расходник — существующий стим-инъектор.⟧
⟦Этап квеста: катушка получена → уведомление; этап 3 остаётся активным, пока
не собраны все три детали.⟧

## Сцена 1.4 — «Руины у периметра» (граница аномалии; Тэмур)

Механики сцены (все — без нового кода):
1. **Зона риска**: сцена поделена маркерами — белая часть безопасна, жёлтая нет.
   Переход за жёлтые метки — осознанное действие («Перешагнуть маркеры») → флаг
   `ch1_in_yellow`: включаются условные фон-эффекты (глитч + виньетка из готовых 15),
   а «Осмотреться» начинает выдавать **искажённые** строки внутреннего голоса
   (сенсорные искажения — канон world-anomalies.md §4). Первое «касание» аномалии —
   без цифр и приборов, чистым ощущением.
2. **Два пути к шине данных** (лежит за жёлтой линией): рискнуть самому — быстро,
   но «поведёт» и Тэмур не одобрит; или поговорить с Тэмуром — он подскажет безопасный
   подход и подстрахует верёвкой (+отношение). Оба пути ведут к цели.
3. **Реюз паттернов 1.3**: «Осмотреться» со счётчиком (свои строки, в двух версиях —
   белая/жёлтая зона), 2 случайные точки обыска.
4. **Посев на будущее**: мини-поручение Тэмура (пропавшие инструменты → гнездо
   падальщика, сцена 1.5) + наводка на само гнездо (закрывает подсказку Кая).

*Локация: руины у западного периметра — остов не то станции, не то посёлка.
Вдоль дороги столбы с белыми метками; глубже, между корпусами, метки жёлтые.
Настроение: тихо, но тишина тут другого сорта — как будто место прислушивается.*

*Нарратив сцены:*

Белые метки на столбах — свежие, подкрашенные. Кто-то следит за ними внимательнее,
чем за собственным сном.
