Статус: **утверждён владельцем 2026-07-07** (вместе с предложениями §4 и §7: инженер —
Лия, торговец — Кай, развилка за деталью №3 — выбор игрока, механика тишины в главе

## 0. Границы глав (решение владельца)

- **Глава 0 (интро)** заканчивается: ГГ и Матис доходят до лагеря-аванпоста,
  ГГ знакомится с несколькими NPC. **Осколка в интро нет.**
- **Глава 1** начинается **через 2 недели после спасения**, там же, на аванпосте.

### Памятка: что подрезать в текущем интро (правки контента — за владельцем)

Сейчас в проекте интро выдаёт Осколок раньше времени. По новым границам:

1. Сцена «Ангар»: убрать выдачу Осколка ур. 1 Матисом (предмет + переменная oskolok=1).
2. Квест «Наладить связи» (награда oskolok=2) — перенести в главу 1 или переделать награду.
3. Финал главы 0 = приход в лагерь + первые знакомства (2–4 NPC, предлагаю: Матис уже есть,
   Лия, Лори (кормит), Джаст (короткое представление лидеру)).
4. Проверить: до получения Осколка переменная oskolok = 0 → шкалы отношений и панель
   фракций в UI не показываются (так и задумано — «ГГ слеп к социальному слою мира»).

## 1. Арка главы: «Чужак получает глаза»

Завязка: две недели ГГ живёт в лагере на птичьих правах. Он — странность: человек без
Mesh, «глухой и немой» для всей инфраструктуры мира (двери, маркеры, инструменты,
расчёты — всё в этом мире говорит через Mesh, а с ним — молчит).

Движение: через первую квестовую цепочку ГГ получает **Осколок** — браслет-суррогат,
собранный руками номадов из деталей. Мир впервые начинает с ним разговаривать.

Смысловая рифма: у всех «второй голос» вшит с рождения — ГГ получает свой снаружи,
как инструмент, который можно снять. Он единственный, для кого интерфейс мира — вещь,
а не часть сознания. (Это же объясняет игроку, почему UI открывается ступенями.)

Финал акта 1 = вручение Осколка ур. 1: в диалогах появляются шкалы отношений —
сюжетно обыграно («теперь ты видишь, как к тебе относятся»).

## 2. Акт 1 — цепочка Осколка (схема владельца, дословно)

**Квест «Второй голос» (сюжетный, этапы через QuestStep):**

1. **К инженеру.** Матис отводит ГГ к Лии Ромеро-Санг (инженер мобильных платформ,
   знакома с гл. 0). Лия загорается идеей собрать браслет-суррогат: «без Mesh ты тут
   как без рук, ушей и половины глаз». Нужны детали — посылает к торговцу.
2. **К торговцу.** Кай Муромото (квартирмейстер: снабжение, инвентарь, переговоры
   с купцами). Скептичен: тратить фонды каравана на чужака? Заказ соберёт,
   но 2–3 недостающие позиции — «найди сам, окрестности богатые».
3. **Исследование окрестностей** (пока Кай собирает заказ). 2–3 сцены вокруг аванпоста:
   - свалка/кладбище техники старого узла (лут, первая деталь);
   - руины у периметра — маркеры номадов класса «Фон» (первое живое знакомство
     с маркировкой аномалий; Тэмур на дежурстве объясняет знаки);
   - источник опасности за деталью №3: дрон-падальщик (бой) или переговоры/обмен —
     развилка на выбор игрока.
4. **Возврат к торговцу.** Кай выдаёт собранный заказ (+ сдвиг отношения, если ГГ
   вёл себя толково; у Кая своя мини-сцена — он пересчитывает всё дважды, травма
   прошлой ошибки).
5. **Возврат к инженеру.** Сборка при игроке (Лия + может подключить Зору/Хани).
   **Вручение Осколка ур. 1.** Включаются шкалы отношений. Финальный бит: Лия
   спрашивает, каково это — слышать мир «снаружи головы».

Опционально в цепочке: короткие реплики-реакции встречных NPC на «человека без Mesh»
(палитра отношений, §4).

## 3. Акт 2 — эскиз (детализируем после утверждения акта 1)

- Задания аванпоста, гейты по репутации `frep_flux` (пороги примерно: 20 → доверенные
  поручения, 40 → допуск к разведке). Репутация растёт от отношений с NPC — значит,
  двигатель акта 2 — личные мини-арки номадов (у каждого NPC в карточке уже есть квест).
- Слух → подтверждение: разведка ловит признаки **новой аномалии в нескольких днях
  от лагеря** (канон: окно прогноза). Первая настоящая экспедиция — климакс главы.
- Кандидат на финальный бит главы: класс зоны оказывается выше прогноза, караван
  принимает решение (остаться/сниматься) — первое большое решение с весом.

## 5. Аномалии главы

1. **Зона у периметра, класс «Фон»** (учебная): маркеры, миражи на краю зрения,
   лёгкие помехи. Функция: научить игрока читать маркировку; источник детали №2.
2. **Прогнозируемая зона (акт 2), заявлена как «Помеха», по факту выше**: климакс.
   Эффекты — сетевые (глушение Mesh) + сенсорные. Детали в проработке акта 2.

Ни один текст главы не намекает на связь аномалий с Архоном (скрытый закон).

## 6. Экономика главы (что нужно в редакторе)

- **Переменные**: флаги этапов цепочки (`ch1_*`), `frep_flux` уже считается автоматически;
  `oskolok` — выставляется в 1 только в финале акта 1.
- **Предметы**: 3 детали-квестника (например: «Резонансная катушка», «Шина данных
  старого стандарта», «Неповреждённый кристалл памяти»), сам «Осколок ур. 1» (квестовый);
  расходники для боя — уже есть.
- **Мобы**: дрон-падальщик (1 шт., по образцу дрона из интро, слабее/другие атаки).
- **Сцены**: лагерь (2–3 экрана: мастерская Лии, склад Кая, общий двор), свалка,
  руины у периметра, край зоны «Фон». Всё строится существующими инструментами
  (bgfx-эффекты для зоны: глитч/виньетка/дрожь), нового кода не требуется.
