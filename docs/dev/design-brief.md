# The Lost Silence — визуальный бриф для дизайн-агента

Назначение файла: единый контекст для работы над внешним видом ИГРЫ (не редактора).
Редактор владельца устраивает — его не трогаем.

---

## 1. Что за продукт

Нарративная игра: диалоги, решения, репутация + лёгкие RPG/idle-механики.
- Платформы: **PC и мобильные браузеры**. Экспорт — один самодостаточный HTML-файл.
- Игровое поле: **16:9, логические координаты 1920×1080**, элементы позиционируются в %,
  шрифты масштабируются через `cqw` (container queries). Всё — DOM/CSS, без canvas/WebGL.
- Ограничение: **никаких внешних ресурсов** (CDN, Google Fonts, внешние картинки).
  Только системные шрифты, inline-SVG и data-URI. Тач-таргеты на мобильных ≥ 44px.

## 2. Лор — минимум для настроения

Год 2670+. После «тихой катастрофы» 2034-го (исчезло 95% человечества — правда скрыта)
миром управляет архитектура, а не власть: мегаструктура **Nexus** сглаживает всё резкое.
У каждого человека с рождения — **Cerebral Mesh**, «второй голос» в сознании: мягкие
подсказки, ноль одиночества. Внутренняя тишина исчезла как явление.
**OldNet** — запретный старый интернет, где лежит правда.

Герой — человек из 2030-х, 600 лет в стазисе. Единственный без Mesh: он **помнит тишину**.
Его интерфейс — **Осколок**, внешний браслет-суррогат: прокачиваясь, он буквально
**открывает игроку UI** (шкалы отношений → панель фракций → подсказки → следы OldNet).

**Ключевая метафора дизайна: стерильный, выверенный, «сглаженный» мир — и трещины в нём.**
UI игры = то, что видит герой через Осколок: слегка «инженерный», полупрозрачный,
неродной для этого мира. Тишина = пустота, воздух, паузы. Правда = тёплые сбои
в холодной системе.

Тон: медитативный, тревожно-спокойный. НЕ киберпанк-неон, НЕ гранж, НЕ милитари.
Ближе: минимализм научной станции, глубина океана, свечение приборов в темноте.

- Панели UI: `rgba(8,13,18,0.92)` + `backdrop-filter: blur`, бордеры `rgba(255,255,255,0.1)`.
- Акцент (Осколок, системное): бирюза `#4fd1c5`, приглушённая `#2a6f68`.

## 4. Существующие элементы игры (полный перечень)

### Сцены
- **Главное меню**: логотип-текст, подзаголовок, кнопка НАЧАТЬ.
- **Локации**: фон (цвет/градиент/картинка) + нарративный текст + кнопки-действия.
  Сейчас фоны — чистые градиенты, картинок нет (будущий арт).

### Диалоговое окно (низ экрана, ~88% ширины)
- Портрет говорящего 2.4em (сейчас — генерируемый SVG-силуэт с инициалами в рамке
  цвета фракции), имя капсом в цвете темы.
- **Шкала отношения** под именем (красный→жёлтый→зелёный градиент по значению 0–100),
  видна с Осколка ур.1.
- Текст реплики; подсказка «▸ дальше»; варианты ответа — список кнопок,
  с Осколка ур.3 у вариантов метки ▲/▼ (улучшит/испортит отношения).

### HUD (только в игровых сценах, не в меню)
- Слева: кнопка ◈ (панель репутации фракций: полосы в цветах фракций, % и «связей X из Y»),
  кружок уровня, полосы HP (красная) и Фокуса (синяя), кнопка 🎒 инвентарь, кнопка 📋 журнал.
- Справа: валюта `⌬ 123` (кредиты).
- Всплывающие уведомления (низ-право): получение предметов (в цвете редкости),
  «▲ Уровень N!», результаты заданий.

### Инвентарь (модальный, ~82% ширины)
- Слева «ЭКИПИРОВКА»: **манекен из 8 слотов** (голова, тело, ноги, обувь, перчатки,
  оружие, гаджет, аксессуар) — сейчас просто сетка 2×4 пунктирных ячеек (место для
  красивого силуэта героя!) + сводка характеристик (уровень/опыт и 8 статов).
- Справа: сетка ячеек (12+, квадратные), иконки предметов (генерируемые SVG: глиф типа
  в рамке цвета редкости — будущий арт), счётчик стека, метка ◈ у квестовых.
- Drag-and-drop на слоты, клик = меню действий (Экипировать/Использовать/Выбросить).
- Кнопки сортировки «по типу / по редкости».

### Бой (полноэкранный оверлей)
- Верх: моб — иконка (SVG-плейсхолдер «злые глаза»), имя, полоса HP.
- Центр: строка-лог событий + **полоса замаха** (жёлтая, убывает) с **зелёной зоной**
  окна реакции — сердце боя, должна читаться мгновенно.
- Низ: полосы HP/FOC игрока + 4 кнопки: ⚔ Атака, ✦ Спецудар (фокус), ➟ Уклон, 🛡 Парировать
  (неактивные — полупрозрачны). Большие тач-таргеты критичны.

### Журнал (модальный, 3 вкладки)
- **📋 Задания**: карточки с типом (суточное/недельное/сюжетное — синий/фиолетовый/жёлтый),
  описанием, строкой «Награда: +25 Кредиты · Стим-инъектор ×2» и кнопкой состояния
  (Забрать награду / Не выполнено / Завтра снова).
- **⚙ Улучшения**: карточки «Дрон-сборщик — ур. 2/5», описание эффекта, кнопка
  «Улучшить · 128 ⌬» (блокируется без денег).
- **⌬ OldNet**: фрагменты для расшифровки; активная — прогресс-бар реального времени;
  готовая — кнопка «Извлечь правду» → **оверлей «кусок правды»**: цитата курсивом
  в рамке акцентного цвета — важнейший эмоциональный момент, заслуживает особого стиля
  (глитч? старая типографика? артефакты данных?).

## 6. Правила для дизайн-агента

1. **Не менять**: механики, тексты, структуру данных, размеры логической сетки 1920×1080.
2. Все стили должны быть выразимы в CSS-in-JS/инлайн-стилях без внешних зависимостей;
   изображения — только inline-SVG или data-URI.
3. Помнить про мобильные: минимум 44px тач-таргеты, читаемость на 360px-ширине
   (в letterbox), никаких hover-only взаимодействий (hover — только украшение).
4. Единый источник цветов: Theme + таблицы фракций/редкостей выше. Новые цвета —
   добавлять осознанно и документировать здесь.
5. Иерархия внимания в бою: полоса замаха > кнопки реакции > всё остальное.
6. Дух: «тишина» = пустота и воздух; «правда» = тёплый сбой в холодной системе;
   «Осколок» = неродной инженерный слой поверх мира. Меньше, но точнее.
