# The Lost Silence — дизайн-документ игровых систем

Статус: утверждённая рабочая версия (2026-07-02).
Источник: видение владельца проекта + предложения Claude. Решения по спорным пунктам делегированы Claude.

## Столпы игры
1. **Диалоги и решения** — ядро. Всё остальное обслуживает историю.
2. **Отношения важнее цифр** — репутация складывается из живых NPC.
3. **Ежедневный ритм** — игрок возвращается за продвижением, накоплениями и кусками правды о 2034.
4. **Редактор — полноправный продукт**: каждая сущность (NPC, предмет, моб, задание) редактируется без кода.

---

## 1. Репутация (двухуровневая)

### Отношения с NPC
- У каждого NPC: `relation` 0–100 и **вес влияния** `weight` 1–10 (лидер ≈ 10, рядовой ≈ 1).
- Отношения меняются эффектами в диалогах (как переменные) и наградами заданий.

### Фракционная репутация (вычисляемая)
```
rep% = Σ(relation_i × weight_i) / Σ(weight_i × 100)   — только по ВСТРЕЧЕННЫМ NPC
```
- Не встреченные NPC не учитываются (появление нового персонажа не обваливает репутацию).
- UI показывает «связей: X из Y» по фракции.
- Сюжетные пороги задаются парой: `реп ≥ N%` **и** `связей ≥ M` — нельзя пройти, задружившись с одним человеком.
- Разные фракции — разные социальные стратегии: в Cavernium решают веса (иерархия),
  в Woodhaven веса у всех равны (община) — задаётся автором в редакторе, не хардкод.

### Осколок (Mesh-устройство ГГ)
Внешний браслет-суррогат Mesh, собранный Матисом (имплант ГГ поставить нельзя — лорная уникальность).
До получения Осколка (первый час игры) игрок **не видит** ни отношений, ни репутации.
Ступени прокачки раскрывают интерфейс:
1. Отношение собеседника в диалоге.
2. Сводная репутация фракций.
3. «Настроение» NPC — подсказки выгодных реплик в диалогах.
4. Следы OldNet — скрытые сцены и лут в экспедициях.

## 2. Характеристики ГГ
`lvl, exp, hp, focus (foc), сила атаки, ловкость, сила крита, шанс крита, защита, выносливость`

- Реген вне боя: hp +1/сек, foc +0.5/сек (реализуется существующими idle-правилами с условием «не в бою»).
- **Ловкость** → ширина окна уклона/парирования в бою (мс). RPG-прогрессия, не ломающая скилл.
- **Выносливость** → базовые ячейки инвентаря; одежда типа «рюкзак» добавляет ещё.
- Рост: базовые значения и прирост за уровень задаются в редакторе (кривая exp: 100 × lvl^1.5).

## 3. Инвентарь и предметы
- **Манекен, 8 слотов**: голова, тело, ноги, обувь, перчатки, оружие, гаджет (Осколок), аксессуар.
- **Ячейки**: базовые 12 + бонус выносливости + бонусы одежды. Drag-and-drop.
- **Редкость**: хлам → потёртый → добротный → высокий → легендарный → **Архонт-класс**.
- **«Квестовый» — флаг, не редкость** (запрет продажи/выброса, любая редкость).
- **Типы**: оружие (клинки/тяжёлое/энергетическое — 3 тактики), одежда/броня, гаджеты,
  расходники (стимы hp/foc, усилители), ресурсы (компоненты узлов, биоматериалы, минералы), квестовые.
- Сортировка: тип / редкость / новизна.
- Предмет — карточка в редакторе: название, иконка (data-URI), тип, слот, редкость, статы,
  цена, описание, флаги. Всё редактируемое.

## 4. Бои (тайминговые, пошаговые QTE)
Решение: **пошаговая модель с окнами реакции**, не реальное время — проще балансировать,
дружелюбно к мобильным (2–3 больших тапа).

- Враг телеграфирует атаку (сжимающаяся полоса/кольцо, базовое окно ~800 мс; ловкость расширяет).
- Реакции: **Уклон** (избежать урон, широкое окно), **Парирование** (окно ~40% от уклона,
  гасит урон + возврат части + даёт foc), **Спецудар** (тратит foc).
- Лорное ограничение: в 2670-м нет войн. Противники — сорванные дроны и автоматика мёртвых
  узлов, агрессивная фауна биосфер Sylvarium, аномалии OldNet-зон, редкие отступники.
  Бои = экспедиции с Flux Nomads (их работа по лору).
- **Mid/end-game (идея владельца)**: Архон начинает создавать «охотников» на ГГ — но только
  когда ГГ становится угрозой, и в зависимости от отношений с Архоном. Общий принцип:
  **мир реагирует на стиль игры** — агрессивный/скрытный/дипломатичный ГГ получает разный отклик.
- Моб — карточка в редакторе: hp, урон, скорость замаха, последовательность паттернов,
  таблица дропа, exp.

## 5. Idle-системы (лорная рамка «работа, пока тебя нет»)
- **Контракты фракций** — фоновый доход кредитов (уже в движке). Уровень контрактов
  открывается фракционной репутацией — связка систем.
- **Дроны-сборщики** — приносят ресурсы; улучшаются за ресурсы/кредиты.
- **Расшифровка фрагментов OldNet** — фрагменты из экспедиций расшифровываются реальным
  временем (часы). Награда — куски правды о 2034 (сюжет!) — главная причина «зайти завтра».

## 6. Задания
- **Суточные** («3 контракта дня» от разных фракций — тянут к разным NPC) и **недельная экспедиция**.
- Рамка суточного сброса — «цикл пробуждений» (ГГ 600 лет жил циклами).
- Задание — карточка в редакторе: условие выполнения (по переменным), награда (эффекты),
  тип сброса (суточное/недельное/сюжетное).

| Редкость «имбовый» | 🔁 переименован в «Архонт-класс» (лор) |
| Редкость «квестовый» | 🔁 стал флагом, не редкостью |

| Мобы редактируемые | ✅ карточки мобов |
| Idle | ✅ контракты + дроны + расшифровка OldNet |
| Суточные/недельные задания | ✅ |
| Месячные задания | ⏸ отложено до объёма контента |
| Торговля/крафт-станки | ⏸ отложено; пока «ресурсы → улучшения» |

## Решения по открытым вопросам (карт-бланш)
- **Портреты NPC**: на старте — стильные генерируемые силуэты с инициалами и фракционным
  цветом; загрузка изображений (data-URI) уже поддержана — арт можно добавить в любой момент.
- **Бой**: пошаговые QTE-окна (см. выше).

## Дорожная карта (каждая фаза = модуль редактора + e2e-тест + коммит)
1. **NPC и отношения** — редактор персонажей (имя, портрет, фракция, вес), привязка реплик
   к NPC, эффекты отношений в диалогах, формула репутации в движке, HUD Осколка со ступенями.
2. **Характеристики + предметы + инвентарь** — редактор предметов, манекен, drag-and-drop, HUD.
3. **Бои** — редактор мобов, боевая сцена QTE, дроп, exp/lvl.
4. **Idle-расширение + задания** — дроны, расшифровка, суточные/недельные, редактор заданий.
5. **Контент интро (Том III)** — капсула → лаборатория → город Flux Nomads, на живых системах.
