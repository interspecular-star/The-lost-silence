# Арт-промпты NPC — The Lost Silence

Промпты для генерации портретов персонажей, написанные заново по описаниям из
`docs/lore/NPC_TLS_v2.docx` — старые арт-промпты из этого файла игнорируются полностью
(они не совпадали с итоговыми картинками, см. `docs/dev/style-bible-higgsfield.md`, раздел 7).
Лор — `docs/lore/TLS_Lore_v2.docx`.

Документ рассчитан на любую модель генерации изображений (Higgsfield, Midjourney, SDXL,
DALL-E и т.п.) — без параметров, специфичных для конкретного инструмента. Для
Higgsfield-специфичных вещей (Soul ID, Moodboard) см. `docs/dev/style-bible-higgsfield.md`.

## Как этим пользоваться

- Каждый блок «Prompt:» самодостаточен — копируется целиком, отдельно от остальных, в любой
  генератор изображений.
- Стиль один на весь проект, во всех промптах: **semi-realistic painterly digital
  illustration — stylized concept-art rendering, visible painterly brushwork, soft matte
  shading**. Не фотореализм, не аниме.
- Футуризм в каждом промпте указан явно и позитивно (мир 2670+, техника как обыденность) —
  вместо негатив-листа со словами «не средневековье / не фэнтези»: упоминание таких слов даже
  в отрицании иногда провоцирует модель нарисовать именно это.
- Этника, пол, возраст, профессия — из канона `NPC_TLS_v2.docx`, не менялись. Фон/местность
  адаптированы под ситуацию каждого персонажа (не общий шаблонный задник для всех).
- Палитра фракции Flux Nomads: burnt orange, charcoal gray, muted sand, dusty blue, olive,
  rust red — у каждого персонажа 2–3 цвета из этого набора, для узнаваемости фракции при
  сохранении индивидуальности.
